FizzBuzz

FizzBuzz is a relatively simple test for coders, sometimes used for job interviews:
- Write a program that prints the numbers from 1 to 100.
- For multiples of three print “Fizz” instead of the number.
- For multiples of five print “Buzz” instead of the number.
- For multiples of both three and five print “FizzBuzz” instead of the number.
FizzBuzz realized in Go.

There are some interesting problems hidden, depending on language and implementation. ;) - Output is either a string or integer -> return values could become problematic
- Some coders tend to nest conditionals -> could lead to mistakes
- Many implementations -> Concatenations, string arrays, print/println, ...
An interviewer could ask for optimizations if the code isn't clean. Then the fun begins. ;)
